# i2x gem

Ruby `gem` for distributed client agents. This enables launching custom **i2x** agents.

## Installation

Add this line to your application's Gemfile:

    gem 'i2x'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install i2x

## Usage

- Check **i2x**'s [client](http://github.com/pdrlps/i2x-client)
